# Number Shooter

A fast-paced HTML5 number shooting game. Destroy waves of numbered blocks, collect power-up gates, and survive as long as you can.

**[Play Now](https://last-tokens.github.io/number-shooter/)** (coming soon)

## How to Play

- **Move** your mouse (or finger on mobile) to aim
- **Shooting is automatic** — position yourself to hit targets
- **Numbered blocks** scroll down toward you. Each hit reduces their number by your bullet power. Clear them before they reach you!
- **Gates** give you power-ups:
  - 🟢 **Multiply gates** (x2, x3) — multiply your shooter count
  - 🟠 **Power gates** (+1, +2) — increase bullet damage
  - 🟣 **Speed gates** — increase fire rate
- **Game over** when any block reaches the bottom

## Controls

| Input | Action |
|-------|--------|
| Mouse / Touch | Aim |
| Arrow keys / A,D | Aim (keyboard) |

## Running Locally

Open `index.html` in any browser. No build step, no dependencies.
