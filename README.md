# Number Runner

A hyper-casual gates runner game. Swipe to choose gates, grow your number, and smash through enemy walls. Inspired by Count Masters, Number Gun, and the gates-runner genre.

**[Play Now](https://last-tokens.github.io/number-runner/)** (coming soon)

## How to Play

- **Swipe left/right** (or drag mouse) to choose your lane
- **Gates** appear in pairs — pick the better one!
  - Green gates: `+10`, `x2` — grow your number
  - Purple gates: `x3` — huge multiplier!
  - Red gates: `-5`, `÷2` — shrink your number (avoid!)
- **Enemy walls** block the track with a number. Smashing through costs that number from yours.
- **Boss wall** at the end of each level — your number counts down as it drills through. Big enough? You win!
- Reach zero = game over.

## Controls

| Input | Action |
|-------|--------|
| Drag / Swipe | Move left or right |
| Arrow keys / A,D | Move (keyboard) |
