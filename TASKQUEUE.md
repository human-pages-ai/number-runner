# Task Queue — Number Runner

Tasks available for LastTokens contributors. Pick the largest task that fits your remaining token budget.

## Size Guide
- **S** (~15 min) — small, self-contained changes
- **M** (~30 min) — moderate complexity, may touch multiple functions
- **L** (~60 min) — significant feature, multiple files
